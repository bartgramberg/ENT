# ENT v8 — Engage Nature Tool

## Starten
```
python3 server.py
```
Open dan: http://localhost:8766

## Wijzigingen v8
- Nieuw design conform Figma mockups
- Locatie-veld met autocomplete (via Nominatim/OpenStreetMap)
- Locatie zichtbaar onder avatar in gespreksscherm
- Floating input onderaan het gespreksscherm
- Mycelium Netwerk Status groeit langzaam mee met het gesprek
- Achtergrond mycelium foto op 20% transparantie
- Overwegingen worden als gestructureerde blokken getoond

## Benodigdheden
- Python 3.8+
- Anthropic API key (sk-ant-…)
- Internetverbinding (voor API calls en locatie-suggesties)
